# ADR 6 - Node and Microsoft Azure

## Context
Node is used for packages and modules. Azure is used for hosting the website and databases.

## Decisions
- The node package Express is used for the backend and routing.
- Microsoft Azure is used to host the website.
- Microsoft Azure is used to host the SQL database.
- The node package mssql is used for the database.
- The body parser package is used to read data from API's.

## Status
Accepted

## Consequences
The website is hosted efficiently.
